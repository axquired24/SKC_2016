## SOLOCUP SKC 2016
SISTEM INFOMRASI PERTANDINGAN *v1.2*

Step :
------
1. Import The Newest Database : db_backup/[bigger_number]skc_solocup.sql
  * [11] Clean Data - Hanya Struktur DB
2. Edit the Config files (lib/aio_config.php)
  * Default :
  * username : root
  * password : root
  * dbname   : skc_solocup
3. Place it on web server
4. Launch!
